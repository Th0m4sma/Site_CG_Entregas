EXERCÍCIO 1

Neste exercício, desenharemos quatro figuras utilizando os modelos apresentados na primeira aula: um robô, uma flor, um carro e um catavento. Cada figura deve empregar as técnicas e estruturas estudadas, como GL_TRIANGLES, GL_TRIANGLE_FAN, entre outras. O objetivo é aplicar os conceitos aprendidos e compreender melhor o pipeline de construção de imagens utilizando WebGL.

EXERCÍCIO 2

Neste exercício, desenharemos quatro figuras utilizando apenas os pontos: um robô, uma flor, um carro e um catavento. Cada figura deve empregar os pontos para formar linhas, triângulos e outras estruturas geométricas estudadas. O objetivo é aplicar os conceitos aprendidos e compreender melhor o pipeline de construção de imagens utilizando WebGL com GL_POINTS.

EXERCÍCIO 3

A partir de um clique do mouse, será traçada uma circunferência centrada na posição clicada, utilizando apenas coordenadas inteiras para identificar os pontos que compõem seu contorno. O objetivo é compreender como o algoritmo de Bresenham permite aproximar curvas suaves usando apenas pontos discretos, o que é essencial para a renderização gráfica em contextos como WebGL.
